# Modern Portfolio Website with Next.js 15

## Overview
Modern portfolio website built with Next.js 15 to showcase projects and professional skills.

## Technical Stack

### Core Technologies
- **Next.js 15**
    - Server-side rendering (SSR)
    - App Router architecture
- **React 19**
    - Server Components
    - Client Components

### Styling & UI
- **TailwindCSS**
    - Utility-first CSS framework
    - Responsive design
- **Framer Motion**
    - Animation library
    - Smooth transitions

### Development Tools
- **TypeScript**
    - Type safety
    - Enhanced development experience

## Key Features

### 1. Performance Optimization
- Automatic image optimization
- Code splitting
- Server-side rendering

### 2. SEO Features
- Meta tags optimization
- Structured data
- Sitemap generation

### 3. User Interface
- Responsive design
- Dark/Light mode
- Smooth animations
